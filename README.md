# 🏀 PS20 Mike Legend Tournament DashboardA highly interactive, dynamic, full-stack tournament tracking board designed to manage and display live basketball game results, team fouls, schedules, and custom rosters in real-time.  This project completely digitizes traditional tournament printouts into a robust, responsive, and completely user-editable web portal.  ## 🚀 Key FeaturesComplete Browser Customizability: Admins can change tournament titles, subtitles, matching dates, scores, fouls, and active lineups directly on the screen without rewriting raw code.  Role-Based Access Control (RBAC):HIGHER ADMINISTRATIVE LEVEL (ADMIN): Grants full write-access to shift game layouts, edit player names, input scores, track foul numbers, and update metadata.  SPECTATOR LEVEL (USER): A clean, crisp read-only view that safely freezes input fields and drop-downs to prevent unauthorized tampering.  Responsive Multi-Column Grid Layout: Built entirely without clunky legacy sidebars, optimizing layout space to display core matchups alongside a beautifully formatted 2-column split roster board.  Native Vector Component Logos: High-fidelity team icons (The Unions' Fish, The Bownes' Medal, The Sanfords' Basketball, and The Barclays' Shield) engineered cleanly as lightweight, lightning-fast inline SVGs.  Smart-Zero Fallback Validation: Numeric score inputs allow instant erasure during text entry without disappearing or throwing compilation bugs, defaulting gracefully back to 0.## 🛠️ Tech StackFramework: Next.js (React)Styling Engine: Tailwind CSSLanguage: TypeScriptState Management: React Hooks (useState)## 📁 Project LayoutPlaintext├── App/

Open http://localhost:3000 in your web browser to view the active application.## 📝 Dashboard User Guide### Swapping Matchup Day SchedulesFlip the upper authorization console select drop-down to HIGHER ADMINISTRATIVE LEVEL.  Click on the team select box in any match card to fluidly swap out who is playing. The vector logo will update automatically!  Increment or decrement the live Scores and Fouls (F:) as the game progresses.When finished, toggle back to SPECTATOR LEVEL to review a clean, locked view of the tournament state.
